# Trash Collector
A 2D arcade game built with Unity where you drive a car, collect trash, and deliver it to the station before time runs out.

## Gameplay
- Drive around and collect trash scattered across the map
- Deliver collected trash to the station to earn points
- Your car can carry up to 5 trash at a time
- Collect speed boosts to move faster
- Pick up clocks to gain extra time
- You have 60 seconds — collect as much as you can!

## Controls
| Key | Action |
|-----|--------|
| W | Move forward |
| S | Move backward |
| A | Turn left |
| D | Turn right |
